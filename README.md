Sistema de Login Simples (HTML, CSS e Python)

Este projeto é um exemplo básico de sistema de **login com usuário e senha**, desenvolvido com **HTML**, **CSS** e **Python (Flask)**.  
O sistema valida o login de apenas **um usuário cadastrado manualmente no código Python**.

Funcionalidades

- Tela de login com usuário e senha
- Cadastrar usuário
- Alteração de senha  
- Validação simples no backend em Python  
- Exibição de mensagem de sucesso ou erro  
- Página inicial acessível somente após login  
- Arquivo CSS separado para estilização

Tecnologias utilizadas

- Python 3
- Flask (micro framework para web)
- HTML5
- CSS3
